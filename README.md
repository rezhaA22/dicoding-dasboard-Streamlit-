## Setup Environment - Anaconda

```
conda create --name env-my python=3.9
conda activate env-my
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal

```
mkdir tugas-saya
cd tugas-saya
pip install -r requirements.txt

```

## Run steamlit app

```
cd ./dasboard
```
